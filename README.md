# @orchestra-mcp/editor

Code and Markdown editor components for Orchestra MCP

Part of the [Orchestra MCP](https://orchestra-mcp.dev) ecosystem.

## Install

```bash
npm install @orchestra-mcp/editor
# or
pnpm add @orchestra-mcp/editor
```

## Usage

```tsx
import { } from '@orchestra-mcp/editor'
```

## Requirements

- React >= 18.0.0
